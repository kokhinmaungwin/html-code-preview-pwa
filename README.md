# HTML Code Preview PWA

A simple Progressive Web App (PWA) that allows you to write HTML code and instantly preview the rendered output below the editor. Features include offline capability, copy code, clear editor, and install as a standalone app.

## Features

- Write HTML code with live preview
- Preview shows rendered output without raw HTML tags
- Copy the code easily with a button
- Clear the editor content quickly
- Offline support via Service Worker (PWA)
- Installable on desktop and mobile devices
- Responsive design for all screen sizes

## How to Use

1. Type or paste your HTML code into the editor area.
2. The preview pane will update automatically with the rendered output.
3. Use the **Copy** button to copy the current code to clipboard.
4. Use the **Clear** button to erase all content.
5. Install the app on your device from the install prompt (if available).

## Installation

Just open the app in a modern browser (Chrome, Edge, Firefox) and use it online or install it for offline use.

## Technologies Used

- ACE Editor for code editing
- Vanilla JavaScript for UI logic and PWA install handling
- Service Worker for offline caching
- HTML5 & CSS3 for layout and styling
